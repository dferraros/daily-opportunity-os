## Section 2: Calendly Page Setup

**This is the first task on March 29, before any connection request is sent.**

Calendly is the pivot point of the B2B track. LinkedIn connections are accepted within 24-72 hours of intent. If there is no booking link ready when that window opens, the opportunity closes. Set this up in 20 minutes. Do not skip.

### Setup Instructions

1. Log into Calendly (calendly.com) — use existing paid plan or activate Essentials at $6/month per calendar
2. Click "New Event Type" → select "One-on-One"
3. Fill in the event details:

| Field | Value |
|-------|-------|
| Event name | 20-min Coffee at PBW — Bit2Me B2B |
| Duration | 20 minutes |
| Location | Bit2Me booth, Carrousel du Louvre, Paris — [booth number when confirmed] |
| Available dates | April 15 and April 16 ONLY |
| Time slots (Paris time) | 9:00, 9:30, 10:00, 10:30, 11:00, 11:30, 14:00, 14:30, 15:00, 15:30 |
| Buffer between events | 10 minutes (prevents back-to-back — you need time to close out notes) |

4. Confirmation email: enable auto-send. When the institutional one-pager PDF is ready (Phase 3), attach it to the confirmation email.
5. Copy your Calendly link. Format: `https://calendly.com/danielferraro/pbw2026-b2b` or similar.
6. Test: book a dummy meeting yourself. Confirm the confirmation email arrives. Verify the time zone displays as CET/CEST (Paris).
7. Paste the final Calendly URL in this doc: **[CALENDLY URL: ___________]**

### Minimum Slots Available

- April 15: 10 slots (9:00–11:30 + 14:00–15:30)
- April 16: 10 slots (9:00–11:30 + 14:00–15:30)
- Total: 20 available slots — supports 8-12 confirmed meetings after no-shows and cancellations

## Appendix: Connection Note Character Count

The connection note below is 192 characters — well within LinkedIn's 300-character limit for connection request notes:

```
Hi [Name], Daniel from Bit2Me — Spain's first MiCA exchange. We're at PBW April 15-16. Would love to connect ahead of the event. Our bank-backed infrastructure might be relevant for [Company].
```

**Do not shorten it.** The specificity ("Spain's first MiCA exchange", "PBW April 15-16", "[Company]") is what drives the 30-50% acceptance rate. Generic notes ("Hi, I'd love to connect") accept at under 15%.
