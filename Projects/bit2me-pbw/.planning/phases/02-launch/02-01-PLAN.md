**Section 1: What It Is**
- An interactive mobile web tool (not native app) for booth engagement at PBW
- Hosted on Bit2Me domain: pbw.bit2me.com/health or bit2me.com/pbw/health
- Purpose: 60-second engagement that generates a personalized "Crypto Portfolio Health Score"
- NOT a financial product. It is a marketing tool that produces a personalized report.
- No login, no email, no PII required on the tool itself

**Section 2: The 5 Questions (one screen per question, multiple choice)**
1. "How long have you been investing in crypto?" — Options: Less than 1 year / 1-3 years / More than 3 years
2. "What is your portfolio split?" — Options: Mostly BTC/ETH / Mix of altcoins / DeFi/NFT heavy
3. "How often do you rebalance?" — Options: Never / Quarterly / Monthly / Weekly
4. "Where do you hold your assets?" — Options: On an exchange / Hardware wallet / Split between both
5. "Are you using a MiCA-regulated exchange?" — Options: Yes / No / I don't know

**Section 3: Scoring Logic**
Define a simple weighted scoring system. Each answer maps to points. Total = sum of all answers, normalized to 0-100.

Provide the exact scoring table:
- Q1: <1yr = 10pts, 1-3yr = 20pts, 3yr+ = 25pts (experience)
- Q2: BTC/ETH = 25pts, Mix = 15pts, DeFi/NFT = 10pts (risk profile)
- Q3: Never = 5pts, Quarterly = 15pts, Monthly = 20pts, Weekly = 15pts (active management, but weekly may signal overtrading)
- Q4: Exchange = 10pts, Hardware = 20pts, Split = 25pts (custody diversification)
- Q5: Yes = 25pts, No = 10pts, Don't know = 5pts (regulatory awareness)

Max possible: 25+25+20+25+25 = 120 raw points. Normalize: Score = (raw / 120) * 100, rounded to nearest integer.

**Section 4: Output Screen**
- Score displayed as a visual dial/gauge (0-100)
- Color coding: 0-40 = red, 41-70 = yellow, 71-100 = green
- 3 personalized text lines based on score bracket and specific answers:
  - "Your Score: [number] / 100"
  - "Your Strength: [generated text]" — examples for each bracket
  - "Your Gap: [generated text]" — examples for each bracket
- CTA button: "Improve Your Score — Trade Fee-Free for 60 Days" -> link to bit2me.com/pbw
- IMPORTANT: Output text must be configurable (stored as JSON or CMS field, NOT hardcoded strings). Diego may require changes to "gap" recommendations. No code deploy to change copy.

**Section 5: Share Mechanic**
- "Share Your Score" button using native Web Share API
- Fallback: Twitter intent URL with pre-populated text: "My Crypto Health Score is [X]/100! Check yours at [link] #PBW2026 @bit2me"
- Score must be shareable without revealing actual portfolio data (only the number)

**Section 6: Analytics**
- On completion: fire CleverTap event `health_score_completed` with properties: {score: number, q1_answer: string, q2_answer: string, q3_answer: string, q4_answer: string, q5_answer: string}
- Track: completions per day, average score, share button clicks
- No PII captured by the tool itself

**Section 7: Technical Requirements**
- Platform: Mobile web (iPhone Safari + Android Chrome required)
- No native app, no download, no heavy JS frameworks
- Works on poor venue Wi-Fi (total page weight under 500KB)
